# Sekretne-Archiwum

**Sekretne Archiwum** to gra przeglądarkowa typu Escape Room, w której gracz zostaje uwięziony w tajnym archiwum pełnym zagadek. Celem rozgrywki jest odnalezienie dowodu konspiracji i ucieczka poprzez rozwiązywanie łamigłówek, zbieranie przedmiotów i otwieranie zamków.

## Główne funkcje

- Interaktywny pokój z obiektami: szafka, sejf, komputer, drzwi
- System zagadek: kody, szyfry, hasła
- Prosty ekwipunek: zbieranie i używanie przedmiotów
- Narracja fabularna odkrywana wraz z postępem
- Klimatyczne komunikaty i notatki
- Animacje i efekty wizualne zwiększające immersję

## Technologie

Projekt realizowany w technologii **Vite + React** na **Typescript**, z wykorzystaniem:

- Hooków do zarządzania stanem gry
- Responsywnego designu
- Modularnej struktury komponentów (łatwa rozbudowa o nowe poziomy, licznik czasu, system zapisu)

## Cel projektu

Stworzenie angażującej gry logicznej z elementami fabularnymi, która łączy atmosferę tajemnicy z interaktywną rozgrywką.
